About Muffin Bot

Muffin Bot is a fun Discord bot designed to enhance your server experience.

With a wide range of features from Hive Stats to moderation tools, Muffin Bot aims to make managing and enjoying your Discord server easier and more engaging.

Features

Moderation Commands

Keep your server secure with these moderation tools:

  • !kick [user] [reason] - Kick a user from the server.
  • !ban [user_id] [reason]-Not required - Ban a user from the server.
  • !mute [user] [duration] [reason] - Mute a user for a specified time.

Custom Per Guild Commands

Customize commands and settings for each server:

These commands require Administrator permissions!

  • !command add [command-name] [command-response] - Adds a custom command with a custom response.
  • !command edit [command-name] [new-command-response] - Edit a commands response.
  • !command remove [command-name] - Removes a custom command you added.

Hive Minigames

Commands for the Hive Minigames server to check statistics! Commands include:

  • /hive-lb - To check the Monthly/Alltime leaders of the games.
  • /hive-stats - To check a players game stats from any game.
  • /hive-overall - To check a players overall stats.

Extra Commands

Some extra commands that could be useful:

  • /time [timezone] - Find what time it is for a certain timezone.
  • !steal [emoji] - Steal a emoji from any server.
